# Employee Leave Management System

This is a simple Employee Leave Management System made using MERN stack.  
Employees can apply for leave and check their leave status.  
Admin can see all leave requests and approve or reject them.

---

## Features

- User signup and login
- Employee can apply for leave
- Employee can see own leave history
- Admin can view all leave requests
- Admin can approve or reject leave

---

## Tech Used

Frontend:
- React.js
- CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- JWT for authentication
